# kylin麒麟网络
## 创建免费账号
浏览器输入
http://faucet.cryptokylin.io/create_account?xxxx
即可创建，并会看到公私钥信息,"xxx"替换为你要创建的账号

## 免费获得Token
浏览器输入
http://faucet.cryptokylin.io/get_token?xxxx
"xxx"替换为你的账号
